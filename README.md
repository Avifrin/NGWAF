# NGWAF

## NGWAF - WAF нового поколения, являющийся базовой программой с открытым исходным кодом и возможность расширения за счёт плагинов сообщества.

### Главной целью является: разработка удобного сервиса для защиты веб-приложения и его программ.

### Текущая версия: α.0.1

### Активный разработчик решения: Avifrin(v26.464.028)
